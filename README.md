## Title
BooksAPI

## Getting Started

git clone the repo

### Prerequisites

A mongo database is required

**Setting up the database**
```
mongo bookAPI < booksJson.js
```

## Running the app
cd into the node-express-mongo-api folder
Install the dependencies
```node
npm install
```
Run the app in development
```node
npm run start
```

## Running the tests
```node
npm test
```
